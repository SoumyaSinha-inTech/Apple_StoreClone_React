
#  Apple Store Clone (React)
A responsive Apple Store-inspired e-commerce frontend built with React. This project replicates the core shopping experience, including product browsing, detailed product pages, a shopping cart, and persistent cart data using Local Storage.

# 🚀 Features
- Browse products by category
- Search products
- Product details page
- Add items to cart
- Increase / decrease item quantity
- Remove items from cart
- Persistent cart using Local Storage
- Cart badge showing total items
- Responsive design

# 🛠️ Tech Stack
- React
- React Router DOM
- JavaScript (ES6+)
- Bootstrap 5
- HTML5
- CSS3
- Local Storage
